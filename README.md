# Bitcoin Price Alert
Bitcoin Price Alert: receive an email when the price of bitcoin is higher or lower than the amount you set.

## Getting Started

- Clone the repository
  ```sh
  git clone https://github.com/travolgi/btc-price-alert.git
  ```
